A new Dataverse plugin enables AI agents to safely interact with enterprise data by providing guardrailed access to tables, columns, relationships, and security. The plugin was demonstrated through three scenarios at Build 2026, showing how developers, analysts, and admins can use natural language to accomplish complex tasks from data modeling to security management.
